About The Page

--------------------------


In this visual representation of population and migration, you can quickly see statistics per country by clicking on the map.


The default year is 2021 (will update to 2022, as soon as the UN population database finalizes migration data for 2022).

You can use the year drop down to select any year back to 2010. This will update the stats in the table, but the map will stay fixed on 2021.


As more databases are added to this visual, more filter buttons will become available at the top.


--------------------------

Data Parameters / Definitions

--------------------------


Population Change

Difference between the population sizes on 1 January of two consecutive years.


Total Net Migration

Net number of migrants, that is, the number of immigrants minus the number of emigrants.


Net Migration Change as a Percent of Population Change

The US, for 2021, shows 55.95%.

Mexico, for 2021, shows -8.25%.


How to Read this Number

In 2021, almost 56% of the US population change is due to migration. The remaining change is from natural population change (live births - deaths).

In Mexico, during the same year, the natural population change offset the migration decrease. Total population is up 0.5%, but net migration is negative.
